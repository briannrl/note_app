# note_app
# Review
<p>
  This is a simple note taking app that can add, update, and delete notes.
</p>

# Stacks
<li>HarperDB: As database</li>
<li>FastAPI: As backend</li>
<li>React.js: As frontend</li>
<li>And a little bit of css</li>
